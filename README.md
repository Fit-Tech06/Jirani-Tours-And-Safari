# Jirani Tours & Safaris - Website

A beautiful, African-friendly website for Jirani Tours & Safaris, showcasing premium Tanzania safari and adventure tour packages.

## About

This website was built for **Said Mwenjuma**, owner of Jirani Tours & Safaris, to showcase their tourism services in Tanzania. The site features an authentic African design with warm colors, modern layout, and comprehensive information about tour packages, national parks, and the Great Wildebeest Migration.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **African-Friendly Design**: Warm color palette inspired by African landscapes
- **Smooth Navigation**: Fixed navbar with smooth scrolling
- **Interactive Elements**: Animated sections, mobile menu, contact form
- **Comprehensive Content**: 
  - Tour packages with pricing and details
  - Information about Tanzania's national parks
  - Detailed migration information
  - Contact form and information

## Publish to GitHub (shareable link)

1. **Install Git** (if you don’t have it):  
   Download and install from [https://git-scm.com/download/win](https://git-scm.com/download/win).  
   During setup, choose “Git from the command line and also from 3rd-party software,” then restart your terminal/Cursor.

2. **Create a new repository on GitHub**
   - Go to [https://github.com/new](https://github.com/new)
   - **Repository name:** e.g. `afroresilience-trust` or `jirani-tours`
   - **Description:** e.g. `Jirani Tours & Safaris – Tanzania safari website`
   - Choose **Public**
   - Do **not** tick “Add a README” (you already have one)
   - Click **Create repository**

3. **Push this folder from your computer**  
   In PowerShell or Command Prompt, run (replace `YOUR_USERNAME` and `REPO_NAME` with your GitHub username and repo name):

   ```bash
   cd "c:\Users\Lenovo\Desktop\Afroresilience Trust"
   git init
   git add .
   git commit -m "Initial commit: Jirani Tours & Safaris website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
   git push -u origin main
   ```

4. **Your shareable links**
   - **Repo (code):**  
     `https://github.com/YOUR_USERNAME/REPO_NAME`
   - **Live site (if you enable GitHub Pages):**  
     In the repo go to **Settings → Pages → Source: Deploy from branch** → branch `main`, folder `/ (root)` → Save.  
     After a minute or two, the site will be at:  
     `https://YOUR_USERNAME.github.io/REPO_NAME/`
